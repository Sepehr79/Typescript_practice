1. npm init
   * initialize package.json file
2. npm install --save-dev lite-server
   * install server
3. npm start
   * start the server

* Basic types
   * number
   * string
   * boolean
   * object
   * arrays
   * tuples
   * enums
   * any

* Other types
   * union        number | string
   * Literal      'text' | 'other text'
   * type aliases   type example = number | string




* Compile in watch mode
`tsc app.ts -w`
